*** ARQUIVOS ***

 - matrix.cpp >> programa que recebe dois parâmetros: <N> <LoopType> e executa uma múltiplicação de dus matrizes NxN utilizando a técnica definida no segundo parâmetro
 - run.py >> script que automatiza o trabalho dividido em 3 etapas: Execução do perf de maneira massiva para gerar os dados brutos. Pré-processamento para transformar os dados em um DataFrame [Pandas required] e por fim Análise com geração dos gráficos pré-definidos.

** Utilização **

 1º - Compilar o arquivo matrix.cpp utilizando g++ ou "make all"
 2º - Verificar o programa compilado utilizando "make test-n", "make test-i" e "make test-u"
 3º - Executar o script run.py para regerar os arquivos da pasta result
